# 🤖 MySchedule Bot (Discord Bot with MongoDB)

บอท Discord สำหรับจัดการตารางชีวิตประจำวัน การเรียน และการบ้าน พัฒนาด้วย Python และใช้ MongoDB ในการเก็บข้อมูลอย่างเป็นระบบ

## ✨ ฟีเจอร์หลัก (Features)
- **📅 ระบบตารางเรียน:** แจ้งเตือนก่อนถึงเวลาเรียนอัตโนมัติ และสามารถใช้คำสั่งดูตารางเรียนรายวันหรือสรุปรายสัปดาห์ได้
- **📝 ระบบจัดการการบ้าน:** เพิ่ม แก้ไข ลบ และดูรายการการบ้านที่ค้างอยู่ (ระบบจะบันทึกลงฐานข้อมูล MongoDB)
- **📊 ระบบโควต้าขาดเรียน:** บันทึกและตรวจสอบโควต้าการขาดเรียนในแต่ละวิชา (แจ้งเตือนหากขาดเกิน 3 ครั้ง)
- **⏰ ระบบแจ้งเตือนกิจกรรม (Reminders):** ตั้งเวลาแจ้งเตือนโปรเจกต์/กิจกรรม และมีระบบลบกิจกรรมอัตโนมัติเมื่อหมดเวลา
- **🌤️ ระบบเช็คสภาพอากาศ:** ตรวจสอบอุณหภูมิและสภาพอากาศปัจจุบันที่บางพระ ชลบุรี
- **🎲 ระบบสุ่มกิจกรรมวันว่าง:** สุ่มกิจกรรมยามว่าง เช่น ต่อโมเดลกันพลา, อ่านมังงะ, เขียนโค้ด Roblox/Minecraft

## 🛠️ เครื่องมือที่ใช้ (Tech Stack)
- Python 3
- discord.py
- MongoDB (Motor AsyncIO)
- โฮสต์ติ้ง: Discloud

## 🚀 วิธีการติดตั้งและรันบอท (Setup & Installation)

1. **ติดตั้งไลบรารีที่จำเป็น:**
   ```bash
   pip install -r requirements.txt
   ```
   
2. **ตั้งค่าตัวแปรแวดล้อม (.env):**
   สร้างไฟล์ .env ในโฟลเดอร์หลัก และใส่ข้อมูลดังนี้ (ไฟล์นี้ถูกซ่อนไว้ใน .gitignore จะไม่ถูกอัปโหลดขึ้น GitHub เพื่อความปลอดภัย):
   ```bash
   DISCORD_TOKEN=ใส่โทเคนบอทของคุณที่นี่
   MONGO_URI=ใส่ลิงก์เชื่อมต่อ_MongoDB_ของคุณที่นี่
   ```

3. **เริ่มต้นการทำงาน:**
   ```bash
   python main.py
   ```
